# Simple Login Page

Open `index.html` in a browser to view the simple login page.

Test credentials (demo):

- Email: `user@example.com`
- Password: `password`

Files created:

- `index.html` — HTML layout
- `styles.css` — basic styles
- `script.js` — client-side validation and demo auth

To run: open the file directly or serve the folder with a static server (optional):

```bash
# using Python 3
python -m http.server 8000
# then visit http://localhost:8000
```
